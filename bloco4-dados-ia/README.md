# Bloco 4 – Análise de Dados e Inteligência Artificial

Projetos desenvolvidos ao final do Bloco 4, utilizando Python, Pandas, Numpy, visualização de dados (Matplotlib/Seaborn) e introdução a IA.
O foco é aplicar raciocínio analítico, tratamento de dados, gráficos, métricas e pequenas automações inteligentes.
