# Bloco 2 – Programação com Python

Projetos desenvolvidos ao final do Bloco 2, utilizando Python para aplicar conceitos fundamentais como variáveis, funções, estruturas de repetição, condicionais, listas, dicionários e leitura de dados.
Os alunos começam a estruturar programas mais robustos e organizados, já seguindo boas práticas.
